# MoneyMind Mobile

App per la gestione delle finanze personali.

## Download

Scarica l'ultima versione dalla sezione [Releases](https://github.com/robertomoccia8-cmyk/moneymind-mobile/releases).

## Installazione

1. Scarica il file `.apk` dalla release più recente
2. Abilita "Installa da origini sconosciute" nelle impostazioni Android
3. Apri il file APK e segui le istruzioni

## Aggiornamenti

L'app verifica automaticamente la disponibilità di nuove versioni.
Vai in **Impostazioni → Aggiornamenti → Verifica Aggiornamenti**.
